## Executive Summary
Added a complete customer support ticket management system. Admins can view, filter, search, reply to, resolve, and close support tickets across 4 categories (billing, listing, technical, general) and 4 priority levels.

1. **AdminSupportView** — Ticket management dashboard
2. **4 KPI cards** — Open, pending, urgent, resolved counts
3. **6 mock tickets** with varied statuses, priorities, categories, and message counts
4. **Priority indicators** — Color-coded dots (urgent=amber, high=blue, medium=gray, low=tertiary) + "URGENT" badge
5. **Status workflow** — Open → Reply (pending) → Resolve → Close
6. **Search + dual filters** — By subject/sender/business, status, priority
7. **Message count** — Chat icon + count per ticket
8. **Timestamp display** — Created date + last reply relative time
9. **Spring entrance** — Each ticket animates in with 0.03s stagger
10. **Urgent border** — Urgent tickets get amber left border (4px)
