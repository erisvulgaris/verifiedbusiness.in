# Cycle 9 Report — SEO: Core Web Vitals + Enhanced Structured Data

**Date:** 2026-07-05 06:00 UTC
**Focus:** SEO — structured data enhancement

## Executive Summary
Created an enhanced SEO structured data system with WebSite schema (enables Google sitelinks search box), Organization schema (enables knowledge panel), FAQPage schema (enables rich FAQ results), and BreadcrumbList schema for any page. These enhance search appearance and click-through rates.

## Features Added
1. **useGlobalSeoSchema** — Injects WebSite + Organization schema globally
   - WebSite: name, url, SearchAction (enables Google sitelinks search box)
   - Organization: name, url, logo, description, foundingDate, areaServed=IN, knowsLanguage (7 languages)
2. **useFaqJsonLd** — Generates FAQPage schema from FAQ items (enables rich FAQ snippets in search)
3. **useBreadcrumbJsonLd** — Generates BreadcrumbList schema from any page's breadcrumb items
4. All schemas clean up on unmount (no duplicate scripts)
5. All use `data-seo` attribute for identification
