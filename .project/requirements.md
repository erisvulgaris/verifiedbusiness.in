# Requirements

## Existing Requirements
1. Premium local business directory for India (28 states, 19,000+ pincodes)
2. Subscription model: ₹999 → ₹4,99,999/year (7 tiers)
3. Ad management: Run FB/Insta ads for businesses based on their plan
4. Admin panel: Full CRUD + analytics + audit logs + support tickets
5. Business owner portal: Manage listing, reviews, ads, subscription
6. Light + dark mode
7. Mobile-first responsive design
8. SEO: JSON-LD structured data, per-view titles

## Newly Discovered Requirements
- Authentication is critical (admin panel is currently open)
- Payment integration needed before any subscription can be sold
- Meta Marketing API integration is the core USP
- Lead generation ("Get Quote") for customer enquiries
- Email notifications for key events (welcome, expiry, review, lead)

## Proposed Requirements
- WhatsApp Business API integration (India's #1 messaging platform)
- AI-powered ad copy generation
- Multi-language support (Hindi, Tamil, Telugu, Bengali, Marathi, Kannada)
- Google My Business sync

## Acceptance Criteria
- A business owner can: sign up → list business → pay subscription → manage listing → run ads → see leads → respond to reviews
- An admin can: approve listings → manage users → view revenue → moderate reviews → manage ad campaigns → handle support tickets
- A customer can: search → find business → view details → call/get directions → write review → get quote
